# Satisfactory Specifications

Satisfactory is an architecture-testing placeholder, not a server implementation.

## Components

- [Placeholder Overview](./components/instance-overview.md)

There are no Satisfactory runtime, settings, port, backup, or lifecycle feature specs.
Adding any such capability requires a new explicit specification and corresponding tests;
Palworld behavior, typed fields, installation/update actions, status probes, save import,
and Overview operations must not be inherited implicitly.
